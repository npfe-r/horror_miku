# **即刻轮回** (Retry Now)

一款使用 Godot 4.6 开发的第一人称恐怖生存游戏。

## 游戏简介

玩家扮演一名被困在夜间学校的学生，学校中有一个可怕的怪物四处游荡。玩家需要探索学校、收集钥匙和道具、解开机关谜题，最终找到出口逃离学校。在探索过程中，玩家需要躲避怪物的追击，利用藏身处和道具来保护自己。

(小心术力口)

## 技术栈

- **游戏引擎**: Godot 4.6
- **开发语言**: GDScript
- **目标平台**: Windows

## 项目结构

```
horror_miku/
├── scenes/              # 场景文件
│   ├── levels/          # 关卡场景
│   ├── objects/         # 可交互物体场景
│   ├── player/          # 玩家场景
│   └── ui/              # UI场景
├── scripts/             # 脚本文件
│   ├── autoload/        # 自动加载单例
│   ├── items/           # 道具系统
│   ├── objects/         # 物体脚本
│   ├── player/          # 玩家控制
│   └── ui/              # UI脚本
├── assets/              # 资源文件 (模型、纹理、音效等)
├── resources/           # 数据资源
└── project.godot        # 项目配置
```

## 操作说明

### 移动控制

| 按键      | 功能      |
| ------- | ------- |
| W/A/S/D | 移动      |
| Shift   | 奔跑      |
| C       | 蹲下 (切换) |
| 鼠标移动    | 视角控制    |

### 交互

| 按键   | 功能      |
| ---- | ------- |
| E    | 交互/拾取   |
| 鼠标左键 | 使用道具    |
| 1-4  | 切换快捷栏道具 |
| 滚轮   | 切换道具    |

## 核心系统

- **第一人称控制器** - 移动、奔跑、蹲伏、噪音系统
- **怪物AI** - 巡逻、警觉、追击、搜索四种状态
- **交互系统** - 统一的交互接口和提示系统
- **门系统** - 普通门、锁门、密码门、卡片门
- **藏身处系统** - 储物柜、桌底、床底等多种藏身方式
- **道具系统** - 关键道具、消耗道具、背包管理

## 开发环境配置

1. 下载并安装 [Godot 4.6](https://godotengine.org/download)
2. 克隆本项目
3. 使用 Godot 打开 `project.godot` 文件
