# **即刻轮回** (Retry Now)

一款使用 Godot 4.6 开发的第一人称恐怖生存游戏。

## 游戏简介

玩家扮演一名被困在夜间学校的学生，学校中有一个可怕的怪物四处游荡。玩家需要探索学校、收集钥匙和道具、解开机关谜题，最终找到出口逃离学校。在探索过程中，玩家需要躲避怪物的追击，利用藏身处和道具来保护自己。

(小心术力口)

## 技术栈

- **游戏引擎**: Godot 4.6
- **开发语言**: GDScript
- **物理引擎**: Jolt Physics
- **渲染器**: Forward Plus
- **目标平台**: Windows

| 按键   | 功能      |
| ---- | ------- |

## 核心系统

### 玩家系统
- **第一人称控制器** - 移动、奔跑、蹲伏、跳跃
- **视角控制** - 鼠标控制视角，支持灵敏度调整
- **体力系统** - 奔跑消耗体力，自动恢复
- **噪音系统** - 不同行动产生不同噪音等级
- **蹲下系统** - 平滑过渡，天花板检测

### 敌人AI系统
- **状态机** - 巡逻、警觉、追击三种状态
- **警觉值系统** - 渐进式警觉，避免立即追击
- **感知系统**
  - 视觉感知：距离、角度、射线检测
  - 听觉感知：声音衰减、噪音检测
- **随机因数** - 正态分布生成，增加不可预测性
- **智能搜索** - 警觉状态检查多个点

### 交互系统
- **统一交互接口** - 可交互物体接口
- **交互提示** - 屏幕中央显示交互提示
- **射线检测** - 2.5米交互范围

### 道具系统
- **背包管理** - 8格背包，4格快捷栏
- **道具类型** - 关键道具、消耗道具、特殊道具
- **道具效果** - 手电筒、噪音诱饵、急救包等

### 其他系统
- **门系统** - 普通门、锁门、密码门、卡片门
- **藏身处系统** - 储物柜、桌底、床底等

## 开发环境配置

1. 下载并安装 [Godot 4.6](https://godotengine.org/download)
2. 克隆本项目
3. 使用 Godot 打开 `project.godot` 文件
