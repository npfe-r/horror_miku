# **即刻轮回** (Retry Now)

一款使用 Godot 4.6 开发的第一人称恐怖生存游戏。

## 游戏简介

玩家扮演一名被困在夜间学校的学生，学校中有一个可怕的怪物四处游荡。玩家需要探索学校、收集钥匙和道具、解开机关谜题，最终找到出口逃离学校。在探索过程中，玩家需要躲避怪物的追击，利用藏身处和道具来保护自己。

(小心术力口)

## 技术栈

- **游戏引擎**: Godot 4.6
- **开发语言**: GDScript
- **物理引擎**: Jolt Physics
- **渲染器**: Forward Plus
- **目标平台**: Windows

## 操作说明

### 移动控制

| 按键      | 功能      |
| ------- | ------- |
| W/A/S/D | 移动      |
| Shift   | 奔跑      |
| C       | 蹲下 (切换) |
| 鼠标移动    | 视角控制    |

### 交互

| 按键   | 功能      |
| ---- | ------- |
| E    | 交互/拾取   |
| 鼠标左键 | 使用道具    |
| 1-4  | 切换快捷栏道具 |
| 滚轮   | 切换道具    |
| I    | 打开背包    |
| Tab  | 关闭背包    |

## 核心系统

### 玩家系统
- **第一人称控制器** - 移动、奔跑、蹲伏、跳跃
- **视角控制** - 鼠标控制视角，支持灵敏度调整
- **体力系统** - 奔跑消耗体力，自动恢复（含恢复冷却）
- **噪音系统** - 不同行动产生不同噪音等级
- **蹲下系统** - 平滑过渡，天花板检测

### 敌人AI系统
- **状态机** - 巡逻、警觉、追击三种状态
- **警觉值系统** - 渐进式警觉，避免立即追击
- **感知系统**
  - 视觉感知：距离、角度、射线检测
  - 听觉感知：声音衰减、噪音检测
- **随机因数** - 正态分布生成，增加不可预测性
- **智能搜索** - 警觉状态检查多个点
- **导航系统** - NavigationAgent3D 路径导航与避障

### 交互系统
- **统一交互接口** - InteractableObject 基类
- **交互提示** - 屏幕中央显示交互提示
- **射线检测** - 2.5米交互范围
- **高亮组件** - 可交互物体边框高亮

### 道具系统
- **背包管理** - 8格背包，4格快捷栏
- **道具类型** - 关键道具、消耗道具、特殊道具
- **道具效果** - 手电筒、噪音诱饵、急救包等
- **拖拽操作** - 背包内拖拽移动/丢弃道具
- **右键使用** - 快捷栏与背包均可右键使用道具

### 门系统
- **普通门** - 平滑开合动画
- **锁门** - 需要钥匙解锁
- **开关控制** - 支持通过开关控制门

### 藏身处系统
- **储物柜/桌底/床底** - 进入/离开藏身
- **被发现概率** - 不同藏身处有不同的被发现几率
- **怪物交互** - 藏身时触发怪物状态变化

### 机关谜题系统
- **开关** - 控制门、灯光等物体
- **物体关联** - 开关可绑定多个目标物体

### UI系统
- **HUD** - 体力条、准星、交互提示、快捷栏
- **背包界面** - 网格布局、道具信息面板、拖拽交互
- **调试UI** - 怪物感知可视化调试工具

### 游戏管理
- **游戏状态机** - 菜单、游戏进行中、暂停、游戏结束、胜利
- **事件总线** - 全局解耦的信号通信系统
- **存档系统** - 内存存档骨架

## 开发环境配置

1. 下载并安装 [Godot 4.6](https://godotengine.org/download)
2. 克隆本项目
3. 使用 Godot 打开 `project.godot` 文件
